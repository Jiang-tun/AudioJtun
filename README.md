# AudioJtun

一款支持多平台视频解析播放的 Android 应用。根据AudioVisual windows桌面应用复刻的App

## 功能特性

### 平台导航
支持以下视频平台快速跳转：
- 腾讯视频
- 爱奇艺
- 优酷
- 哔哩哔哩
- 芒果TV

### 视频解析
集成多个第三方解析接口，支持解析播放各大平台视频：
- 虾米视频解析
- 七七云解析
- Player-JY
- 789解析
- 极速解析
- 冰豆解析
- 973解析
- CK
- 七哥解析
- 夜幕
- 盘古
- playm3u8
- 芒果TV1 / 芒果TV2
- HLS解析

### 美韩日剧
内置多个美韩日剧资源网站：
- 影巢movie
- 猴影工坊
- 茉小影
- 网飞猫

### 浏览器功能
- 后退 / 前进 / 主页导航
- URL 地址栏直接输入
- 刷新重载页面
- 折叠式侧边栏

## 使用方法

1. **选择平台**：从左侧菜单选择要访问的视频平台
2. **粘贴链接**：在地址栏粘贴视频页面链接
3. **选择解析接口**：选择一个可用的解析接口
4. **点击解析**：点击 PARSE! 按钮开始解析播放

## 技术架构

- **平台**：Android
- **核心组件**：WebView
- **解析方式**：第三方视频解析接口
- **UI 风格**：暗色主题
